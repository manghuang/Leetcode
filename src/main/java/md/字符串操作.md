# 一个字符串，不能重排序
## 最长上升子序列
    1. 动态规划：
    状态：令dp[i]表示[以num[i]为最后一个数，可以选出的上升序列的最大数目]
    复杂度O(n2)
    2. 贪心算法：
    状态：d[i]表示长度为 i 的最长上升子序列的末尾元素的最小值
    复杂度O(nlogn)
